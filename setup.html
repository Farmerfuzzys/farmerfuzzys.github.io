Fire Dispatcher - Setup

Willkommen zum Fire Dispatcher Setup-Tool. Dieses Werkzeug hilft Ihnen, Ihre Firestore-Datenbank einzurichten.

Wichtig: Beachten Sie die Anleitung in der README.md für die vollständige Setup-Anleitung.

Schritte zum Setup:

  1. Lesen Sie die README.md sorgfältig
  2. Erstellen Sie ein Firebase-Projekt auf firebase.google.com
  3. Aktivieren Sie Authentication (Email/Passwort)
  4. Erstellen Sie eine Firestore Database
  5. Erstellen Sie Benutzer in der Authentication Console
  6. Fügen Sie Rollen-Informationen in Firestore hinzu (siehe README.md)

Benutzer manuell hinzufügen:

Sie können Benutzer auch direkt in der Firebase Console hinzufügen:

  1. Gehen Sie zu Build → Authentication
  2. Klicken Sie auf Benutzer erstellen
  3. Geben Sie Email und Passwort ein
  4. Kopieren Sie die UID des neuen Benutzers
  5. Gehen Sie zu Firestore Database und erstellen Sie ein Dokument in der users Collection
  6. Setzen Sie die UID als Document ID und fügen Sie die Rolle hinzu
